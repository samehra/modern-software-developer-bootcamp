# Hands-On Projects

All 10 projects from the Modern Software Developer Bootcamp in one place.

Each folder contains a `PROMPTS.md` — use these prompts with Claude Code to build the project yourself. Once you're done, compare your result against the reference solution in the corresponding `day-X/projects/` folder.

## How to Use

1. Pick a project folder
2. Read its `PROMPTS.md`
3. Open a terminal in an empty directory and start Claude Code: `claude`
4. Run the prompts in order
5. Check your result against the reference solution at `day-X/projects/<project>/solution/`
